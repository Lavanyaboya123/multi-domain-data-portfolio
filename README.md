# Lavanya's Multi-Domain Data Analysis Portfolio

This repository contains my end-to-end data analysis projects using Python (pandas, seaborn, matplotlib).  
Each project includes data cleaning, statistical analysis, visualizations, insights, and recommendations.

## Completed Projects

### Project 1: Retail – Supermarket Sales Analysis
- **Objective**: Analyze sales patterns, best-selling products, and customer behavior.
- **Key Findings**:
  - Total revenue: $322,966.75
  - Top categories: Food & beverages, Fashion accessories, Electronic accessories
  - Females contribute ~51% of sales
- **Files**:
  - [Notebook](supermarket_sales_analysis.ipynb)
  - [Visualizations](supermarket_visuals.png)
  - [PDF Report](supermarket_sales_analysis.pdf)

### Project 2: Education – Student Performance Analysis
- **Objective**: Examine exam scores, pass rates, subject performance, and factors like test prep, parental education, lunch type.
- **Key Findings**:
  - Average scores: Math ~66, Reading ~69, Writing ~68
  - Pass rate (≥60% average): ~89%
  - Test preparation course boosts scores by ~10–15 points
- **Files**:
  - [Notebook](student_performance_analysis.ipynb)
  - [Visualizations](student_performance_visuals.png)
  - [PDF Report](student_performance_analysis.pdf)
